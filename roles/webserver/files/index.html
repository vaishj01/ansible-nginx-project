Web Server Automation Using Ansible

Automated Deployment with Jenkins, GitHub and AWS EC2

Project Overview

This project demonstrates automated Linux web server deployment using Ansible. The complete server configuration process is automated through Infrastructure as Code practices.

Technologies Used

  • AWS EC2
  • Ansible
  • Jenkins
  • Git & GitHub
  • Nginx
  • Linux (Ubuntu)

Automated Tasks

  • Create Application Directory (/opt/webapp)
  • Copy Website Files
  • Install Nginx
  • Configure Web Server
  • Start and Enable Services
  • Verify Port 80 Availability
  • Automated Service Restart using Handlers

Deployment Status

✅ Deployment Successful

Website deployed automatically through Jenkins Pipeline and Ansible Playbook.
